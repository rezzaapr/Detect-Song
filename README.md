<h1>installation</h1>

* ``git clone https://github.com/rezzaapr/whats-the-lyric`` 
* ``cd whats-the-lyric`` 
* ``npm i``

<h1>Running App</h1>

* ``node app.js``
* After Type ``node app.js`` Output will be like this:
```
âžœ whats-the-lyric node app.js
App running on *: 8080
```
* Open On Browser ``http://localhost:8080/detect?={query}``
